### 処理詳細（Action.execute フロー）

| No. | 処理内容 | 関連DAO | 備考 |
| --- | --- | --- | --- |

| サブシステム | カレンダー管理 |
| 版数 | 1.0 |
| 作成日 | 2026/05/23 |

### プログラム概要

| 区分 | 項目名 | 型 | Formプロパティ | 必須 | 備考 |
| --- | --- | --- | --- | --- | --- |
| 入力 | 年度 | String | calYear | ○ |  |
| 入力 | 日付範囲(from/to) | String | calDateFrom/To | ○ |  |
| 入力 | 休日種別 | String | calType | ○ |  |
| 入力 | 振替休日/出勤日 | String | transferFrom/To |  |  |

### 処理詳細（Action.execute フロー）

| No. | 処理内容 | 関連DAO | 備考 |
| --- | --- | --- | --- |
| 1 | 月別カレンダー描画: 日付計算→曜日→週割り→7列×5-6行table |  |  |
| 2 | 休日セル色分け: 法定=赤class、会社指定=青class、停止=黄class |  | CSSクラス |

| 区分 | 項目名 | 型 | Formプロパティ | 必須 | 備考 |

| 入力 | 単価 | Integer | price |  |  |

### 処理詳細（Action.execute フロー）

| 2 | 登録: 適用設備のツリー描画（設備種別→設備名の2階層checkbox）→保存 |  |  |
