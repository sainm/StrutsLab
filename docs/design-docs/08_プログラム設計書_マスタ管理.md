## 点検項目マスタ登録・編集 (PG-MST-CHK-002)

| 項目 | 値 |
| --- | --- |
| プログラムID | PG-MST-CHK-002 |
| プログラム名 | 点検項目マスタ登録・編集 |
| サブシステム | マスタ管理 |
| 版数 | 1.0 |
| 作成日 | 2026/05/23 |

### プログラム概要

点検項目テンプレートの3階層ツリー編集。大分類→中分類→個別項目の動的行追加/削除（Submit→再描画）。Indexed + Nested Properties。

### 入出力定義

| 区分 | 項目名 | 型 | Formプロパティ | 必須 | 備考 |
| --- | --- | --- | --- | --- | --- |
| 入力 | テンプレート名 | String | tmplName | ○ |  |
| 入力 | 設備種別 | String | eqpType | ○ |  |
| 入力 | 点検種別 | String | checkKind | ○ |  |
| 入力 | 大分類[{n}].name | String[] | cat1Names |  | Indexed |
| 入力 | 中分類[{n}][{m}].name | String[][] | cat2Names |  | Nested |
| 入力 | 項目[{i}].* | ItemDto[] | items |  | Indexed |

### 処理詳細（Action.execute フロー）

| No. | 処理内容 | 関連DAO | 備考 |
| --- | --- | --- | --- |
| 1 | 動的行操作の判定（addRow/delRow/save） |  |  |
| 2 | 行追加: Formの配列サイズ+1→新しいindexに空データ→INPUT forward |  | 既存行indexは維持 |
| 3 | 行削除: 指定indexを配列から除去→残りをindex=0から詰め直し |  | 配列再構築 |
| 4 | 保存: 3階層構造をトランザクション保存 | ChkItemDao | 大分類→中分類→項目の順 |

### 入力検証ルール

| No. | 対象項目 | 検証ルール | エラーメッセージ（ja） |
| --- | --- | --- | --- |
| 1 | tmplName | 空不可、重複不可 | テンプレート名は必須です / 同名が存在します |  |
| 2 | cat1Names[{n}] | 空不可 | 大分類名を入力してください |  |
| 3 | items[{i}].name | 空不可 | 項目名を入力してください |  |

### 状態遷移

| 遷移元 | 遷移先 | トリガー |
| --- | --- | --- |
| 表示 | 行追加→再表示 | ＋ボタン |
| 表示 | 行削除→再表示 | 削除 |
| 表示 | 保存→一覧 | 保存 |
| 表示 | 一括追加画面 | 一括項目追加 |
